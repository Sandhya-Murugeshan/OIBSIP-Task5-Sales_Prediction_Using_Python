# OIBSIP-Task5-Sales_Prediction_Using_Python

#  Oasis-T5: Sales Prediction using Machine Learning

##  Project Overview
Sales prediction involves estimating how much of a product customers will purchase based on advertising spend across different platforms.

In this project, a Machine Learning model is built to predict product sales using advertising budgets for TV, Radio, and Newspaper.

---

##  Objective
- Analyze the relationship between advertising spend and sales  
- Build a regression model to predict sales  
- Evaluate the model performance  

---

##  Dataset
The dataset contains the following features:

- **TV** – Advertising budget spent on TV  
- **Radio** – Advertising budget spent on Radio  
- **Newspaper** – Advertising budget spent on Newspaper  
- **Sales** – Product sales (target variable)  

 Dataset Link: https://www.kaggle.com/datasets/bumba5341/advertisingcsv

---

##  Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

##  Project Workflow

### 1. Data Loading
- Load dataset using Pandas  
- Inspect data structure  

### 2. Data Preprocessing
- Check for missing values  
- Clean dataset if required  

### 3. Exploratory Data Analysis (EDA)
- Pairplot visualization  
- Correlation heatmap  

### 4. Feature Selection
- Input Features: TV, Radio, Newspaper  
- Target Variable: Sales  

### 5. Model Building
- Train-test split  
- Apply Linear Regression  

### 6. Model Evaluation
- Mean Squared Error (MSE)  
- R² Score  
---

## 📈 Results
- The model predicts sales based on advertising budgets  
- TV advertising shows a strong influence on sales

This project demonstrates how machine learning can be used for sales prediction. Businesses can use such models to optimize advertising strategies and improve decision-making.
